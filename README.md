# AstarVisualization
使用opencv展示A*算法整个搜索过程
## 依赖
--opencv4.7.0
```shell
安装参照 https://blog.csdn.net/weixin_43863869/article/details/128552342
```
## 编译运行
```shell
git clone https://github.com/dongtaihong/AstarVisualization.git
cd AstarVisualization 
mkdir build && cd build
cmake .. && make
./astar
```
